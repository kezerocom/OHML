> #### Example

<sub>OHML</sub>

```sh
<html:
  <head:
    <title: Hello World Page>
    <meta name="description" content="Our first hello world page":>
    <link rel="stylesheet" href="styles.css":>
  >
  <body:
    <p class="message": Hello World>
  >
>
```

<sub>HTML</sub>

```html
<!doctype html>
<html>
    <head>
        <title>Hello World Page</title>
        <meta name="description" content="Our first hello world page">
        <link rel="stylesheet" href="styles.css">
    </head>
    <body>
        <p class="message">Hello World!</p>
    </body>
</html>
```
