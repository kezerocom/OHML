> #### Example

<sub>OHML</sub>

```cs
<html:
  <head:
    <title: hello world page>
    <meta name="description" content="hello world page description":>
    <link rel="stylesheet" href="styles.css":>
  >
  <body:
    <p class="message": hello world!>
  >
>
```

```html
<html>
    <head>
        <title>hello world page</title>
        <meta name="description" content="hello world page description">
        <link rel="stylesheet" href="styles.css">
    </head>
    <body>
        <p class="message">hello world!</p>
    </body>
</html>
```
